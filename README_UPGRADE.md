## 🚀 部署指南

### 快速部署（5 步）
```bash
# 1. 备份生产环境
tar -czf htf_backup_$(date +%Y%m%d).tar.gz htf/

# 2. 上传升级后的文件（保留 data/ 和 userdir/）

# 3. 设置权限
chmod -R 755 htf/
chmod -R 777 htf/{data,userdir,session}/

# 4. 验证 PHP 版本
php -v  # 应显示 PHP 7.3+ 或 8.x

# 5. 访问论坛首页，测试功能
```

### 回滚方案
```bash
# 如遇严重问题，立即回滚
tar -xzf htf_backup_YYYYMMDD.tar.gz
```
